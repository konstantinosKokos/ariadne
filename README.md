# Ariadne

[![CI](https://github.com/konstantinosKokos/ariadne/actions/workflows/ci.yml/badge.svg)](https://github.com/konstantinosKokos/ariadne/actions/workflows/ci.yml)

[WIP]
A Python library for defining LLM pipelines as typed state transition systems. Built because I mostly hate working 
with Lang* frameworks and I want transparency, observability and verification built-in (to the extent that the language 
and the technology allow, that is).

The codebase is minimal and should be straightforward to understand.
